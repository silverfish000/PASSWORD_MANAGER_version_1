# PASS WORLD — Version 1

Cette première version de mon gestionnaire de mots de passe est volontairement imparfaite et désordonnée.

Elle représente mes débuts dans l’apprentissage du langage et du développement en général.  
J’ai choisi de laisser ce projet tel quel afin de pouvoir comparer mon niveau actuel avec mes futures versions et mesurer concrètement mon évolution au fil du temps.

Ce dépôt fait donc partie de mon parcours d’apprentissage et sera complété par des versions plus abouties à l’avenir.
